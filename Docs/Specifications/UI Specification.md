# UI Specification

## Priorities

- readable outdoors;
- fast manual entry;
- large tap targets;
- minimal typing;
- clear safety warnings;
- transparent estimates;
- no hidden automatic decisions.

## Primary Navigation

- Home
- Evaluate Item
- Current Load
- Yard Prices
- Item Library
- Receipts
- History
- Settings

## Home Actions

- Evaluate Item
- Add Measured Material
- Open Current Load
- Update Yard Prices
- Record Receipt

## Estimate Display

Show:

- low;
- likely;
- high;
- selected yard;
- rates used;
- recovery level;
- missing-price warnings;
- important safety warning;
- whole versus stripped comparison when available.

## Unit Preference

User can switch between:

- US customary
- metric

The unit preference changes input defaults and display only.
