# Scrap Helper Knowledge Base

This directory contains the curated domain knowledge used to research, validate, and eventually seed the Scrap Helper application.

## Purpose

The knowledge base separates:

- researched reference information;
- measured personal results;
- yard-specific classifications;
- unresolved claims;
- application seed data.

It is not automatically treated as verified truth. Every numeric or safety claim should carry a source, confidence level, or measurement reference.

## Directory Structure

### Categories
Canonical scrap categories and mappings to scrapyard terminology.

### Materials
Definitions, preparation rules, identification guidance, and price-history notes for yard materials.

### Components
Recoverable parts found inside larger objects.

### Objects
Whole-item templates such as microwaves, filing cabinets, grills, appliances, computers, and audio equipment.

### Safety
Hazards and handling guidance.

### Scrapyards
Yard-specific categories, policies, preparation requirements, and questions.

### Pricing
Rate-source rules, price history, receipts, and reconciliation.

### Research
Source register, claim tracking, measurements, captures, and research standards.

### Images
Reference images that are legally reusable or personally created.

## Evidence Classes

1. Manufacturer specification
2. Government or recognized safety authority
3. Scrapyard policy or receipt
4. Direct measurement
5. Controlled teardown or batch measurement
6. Multiple consistent field reports
7. Single community claim
8. Unresolved estimate

## Core Rule

Generic estimates help start the system. Actual local receipts and measured teardowns improve it. Weak claims must never silently become high-confidence seed data.
